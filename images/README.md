# Thư mục Images

Đặt ảnh của bạn vào thư mục này.

## Yêu cầu ảnh:

### Ảnh Profile (About Section & Header)
- **Tên file**: `profile.jpg` hoặc `profile.png`
- **Kích thước khuyến nghị**: 600x600px hoặc lớn hơn (tỷ lệ 1:1 - hình vuông)
- **Format**: JPG, PNG, WebP
- **Chất lượng**: Ảnh rõ nét, độ phân giải cao
- **Vị trí sử dụng**: 
  - Header/Navigation bar (avatar nhỏ)
  - About Section (ảnh profile lớn)

### Ảnh Strategy (Strategy Section)
- **Tên file**: `strategy.jpg` hoặc `strategy.png`
- **Kích thước khuyến nghị**: 800x800px hoặc lớn hơn (tỷ lệ 1:1 - hình vuông)
- **Format**: JPG, PNG, WebP
- **Chất lượng**: Ảnh rõ nét, độ phân giải cao
- **Vị trí sử dụng**: Bên phải của phần "Phương Pháp Giao Dịch" trong Strategy Section

### Cách thêm ảnh:
1. Đặt ảnh của bạn vào thư mục này với tên tương ứng:
   - `profile.jpg` hoặc `profile.png` - cho avatar và about section
   - `strategy.jpg` hoặc `strategy.png` - cho strategy section
2. Website sẽ tự động hiển thị ảnh

### Lưu ý:
- File ảnh sẽ được tối ưu hóa tự động khi build
- Nên sử dụng ảnh có kích thước hợp lý (< 1MB) để tối ưu tốc độ tải trang
- Nếu không có ảnh strategy, website sẽ hiển thị icon placeholder
